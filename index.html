Projet encadré

Le projet s'effectue dans le cadre du cours Programmation et Projet Encadré (PPE1). Le code de ce cours est L7TI005-P. Ce projet est dans la continuité de la formation pluriTAL enseigné sur un ensemble de trois universités qui sont la Sorbonne Nouvelle, l'INALCO et Nanterre Université.

Présentation du projet :

L'objectif de ce projet est de faire aux étudiants un travail de groupe, ce qui demande de savoir s'organiser correctement entre les membres. Ce sont les professseurs Yoann Dupont et Pierre Magistry qui assurent l'encadrement du projet et les notions de cours en programmation bash.

Le projet encadré :

Le projet encadré se fait en groupe de trois étudiants. Chaque étudiant se devra de maitrisé au moins une langue différente de celle de ses camarades de groupe. Il etait bénéfique de former un groupe de trois personnes provenant de trois université différentes. Ceci afin que chacun puisse apporter des connaissances qui lui sont propre aux autres.

Dans le cadre du projet, chaque groupe choisi un terme qu'il serait interessant d'étudier dans les trois langues. Les utilisations de ce mot seront ensuite comparées par le biai de l'analyse de contexte dans des sites internets. Pour chaque langue, il est demandé d'analyser une cinquantaine de liens.

Comment est-il construit le projet ?

Voici les étapes principales :

  • Création d'un tableau html à l'aide de scripts bash qui permet de : Parcourir les urls avec la commande curl dans bash ; Retrouver les éléments intéressants tels que le code HTTP, l'encodage de la page ; Fait les aspirations des URLs ; Créé le dump des aspirations ; Trouve les occurrences du mot étudié ; Inscrit les contextes des occurrences dans un fichier texte.
